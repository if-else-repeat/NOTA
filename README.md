# NOTA — India's Election Universe

An interactive educational web experience designed to make India's electoral system easier to understand. Through timelines, historical context, election processes, and democratic institutions, the project presents election education in an engaging and accessible format.

## Purpose

The goal of this project is to simplify election education and promote civic awareness through interactive storytelling and visual learning.

## Background

This project was created for the **PromptWars Virtual Hackathon** under the **Election Process Education** challenge track. The objective was to build an interactive experience that helps users understand election processes, timelines, and democratic participation in a clear and accessible way.
